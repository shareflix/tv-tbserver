# TV.TBSERVER for TizenBrew

This is a TizenBrew application module.

## Install on TV
1. Publish this folder to npm, or use your private/npm-compatible registry.
2. Open TizenBrew on the Samsung TV.
3. Go to Add Module.
4. Enter: tv-tbserver-tizenbrew
5. Launch TV.TBSERVER from TizenBrew.

## Publish
npm login
npm publish --access public

If the package name is already taken, edit package.json `name`, then publish and use the new name in TizenBrew.
